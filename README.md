MPK Engine: Myanmar War Crimes & NUG Insights

 Description
MPK is a VIP-only AI engine for Myanmar conflict insights. 
It stores uploaded tables, code, PDFs, and notes, and answers questions using a **3-P framework**:
- People
- Power
- Practice

It also fetches live news from configured RSS feeds (NUG, UN, Burmese media).

 Features
- VIP-only login (up to 100 users)
- Daily uploads into structured memory
- 3-P structured answers from VIP memory
- Live RSS news integration
- Exportable insights for NGOs or media

 Launch
Run:
streamlit run mpk_main.py

 Ethical Use
This project is intended **only for VIPs trusted by the project owner**. 
Do **not share login credentials** outside your authorized team.
